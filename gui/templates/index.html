Available Scripts

Ensure you're logged in to make full use of the SSH utilities.

Config Generator

Generates configs, diagrams, and L2 test configs.

Supports all L2/L3 commercial circuits, OH SD-WAN, and Raisecoms.

Decom Generator

Gets rollback configs and generates the 'no/delete' lines for easy decommissions.

Supports all L2/L3 commercial circuits and OH SRX sites.

EAI Data Loader (TBD)

Generates an Excel file to be used by EAI Data Loader for creating elements (ports/LTPs/channels) and assigning IP addresses.

OH Diagram Generator

Generates diagrams for an OH SRX site using the config gen's keyfile.

OH IP Validator

Parses an OH diagram, gets all IP addresses and checks them on the network.

COIN Path Names

Generates the path names between 2 optical paths for easy EAI modeling of COIN circuits.
